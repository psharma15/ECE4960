This folder contains 4 solvers: 
1. jacobiRc.m: Takes matrix in row compressed format and applies Jacobi iteration
2. jacobiRcD.m: Takes matrix in row compressed format and applies Jacobi iteration after diagonal domination of (A+Epsilon*I)x = b
3. gaussSeidel.m: Takes matrix in row compressed format and applies Gauss Seidel iteration
4. gaussSeidelRcD.m: Takes matrix in row compressed format and applies Gauss Seidel iteration after diagonal domination of (A+Epsilon*I)x = b
